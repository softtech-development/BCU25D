# Lektion 10

I denna lektion ska vi lära oss allt om Objekt Orienterad Programmering med JavaScript

Vi kommer att gå igenom hur vi kan använda funktioner som mallar för objekt. För att sedan gå vidare hur vi skapar
egenskaper(keys) för att lagra tillstånd(data) i våra objekt. Samt hur vi kan definiera beteende(funktionalitet) i våra objekt.

Därefter ska vi se hur vi kan använda Prototyping för att utöka en funktion(objekt) med mer funktionalitet för att slutligen se hur vi kan arbeta med arv i JavaScript.

Nästa moment i lektionen är hur vi kan använda E6 klasser som egentligen endast är _syntactic sugar_ för vad vi lärt oss redan men som gör det lättare att arbeta med objekt.
