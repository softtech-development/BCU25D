# Lektion 7

I denna lektion ska vi fortsätta att arbeta med manipulering av DOM.
För att börja så finns det en färdiga applikation som start projekt. _westcoast-cars-starter_ som innehåller all html och css.
Det som saknas är JavaScript koden för att driva logiken i applikationen.
Dessutom ska vi göra en ny djupdykning i lite mer avancerad JavaScript.

Vad fokus kommer att vara är att:

- Hantera arrayer
  - Kopiera och välja ut vad vi vill använda ur en array
  - Använda _spread_ operatorn för att kopiera arrayer
- Utnyttja _destructuring_ av objekt
- Skapa en struktur som är lätt att hantera oavsett storlek på applikation
- Titta på några olika sätt att skapa html element och attribut.
- Arbeta med formulär och FormData objektet i JavaScript.
