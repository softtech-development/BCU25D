# Lektion 9 TDD och OOP

### Test Driven Development

Under lektion ska vi gå igenom introduktion till Test Driven Development(TDD) och varför det är så viktigt att kunna arbeta enligt TDD principerna
Vi kommer att titta lite på olika testramverk men kommer att använda _vitest_ för det är ett testramverk som används
Test driven utveckling betyder att vi skriver tester först och sedan skriver vi kod som ska klara testerna.
Ett annat uttryck är att testerna driver koden framåt.

### Objekt Orienterad Programmering

Vi kommer även att bekanta oss med paradigmen OOP(Object Orientated Programing).
Vilket handlar om hur vi kan använda JavaScript för att skapa logik i våra applikationer som är återanvändbar och som även går att vidareutveckla via _arv_.

Vi kommer att gå igenom det som i JavaScript kallas för _Prototyping_ så att vi förstår hur JavaScript egentligen hanterar klasser och objekt.
Därefter ska vi ta och titta på ett _syntactic sugar_ ES klasser.
