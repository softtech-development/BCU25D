### Mål
Nu ska vi avsluta vår westcoast-cars applikation och Supabase implementeringen.

### Komma igång
Kom ihåg att i roten på applikationen köra kommandot `npm install` för att installera alla beroende som vår applikation behöver
För att starta applikationen kör kommandot `npm run build`.
